在学习数组的时候我们来回想一下变量的学习过程。变量声明、赋值、初始化。

数组的赋值与访问

对数组的赋值和访问都可以通过下标运算符来实现。例如对于数组 a[5] ，它的数组元素为 a[0]、a[1]、a[2]、a[3]、a[4]。

注意：数组的下标是从 0 开始的，而不是 1 。a[2]下标为2，表示a[2]是首个元素之后的第2个元素。

前面的学习中数组的元素都是 int 型或者 double 型，数组的元素也可以是数组本身。以数组作为元素的数组是二维数组，以二维数组作为元素的数组是三维数组，依次类推。
